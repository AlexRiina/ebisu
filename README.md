## Ebisu Standalone version
This branch contains polyfills for all Numpy and Scipy functions, for use in, e.g., embedding in other applications. All tests continue to pass, and the difference between the full Ebisu implementation (using Scipy for, e.g., `betaln`) and this implementation are close to machine precision.
